## Como utilizar

EN CONSTRUCCION

### Requisitos previos

* nodejs
  https://nodejs.org/en/download/
* react
  ```sh
  npm install react@latest -g
  ```

### Instalacion

1. Clonar el repositorio
   ```sh
   git clone https://github.com/Clyzer/naves-espaciales.git
   ```
2. Instalar paquetes desde NPM
   ```sh
   npm install react@latest -g
   ```
3. Inicia la aplicación mediante
   ```sh
   yarn start
   ```
